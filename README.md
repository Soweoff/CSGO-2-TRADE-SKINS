Projeto da Avaliação 1 de Dispositivos Móveis.

Este projeto tem o intuito de listar itens do jogo Counter-Strike 2 e seus respectivos valores. Esse sistema me ajudará os traders de itens de Counter-Strike 2. Esse Aplicativo ajuda no cálculo de para compra e venda e sua respectiva % de lucro.
